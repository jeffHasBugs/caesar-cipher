# caesar-cipher
Potentially insecure, barebones but working caesar cipher program.
Program takes a string of text to shift it by a specified number and returns the encrypted/decrypted text. Shifts alphabets only.

Example
````
Encrypt[e] / Decrypt[d]?
e
Message :
fOo BAr/?       
Key :
10
pYy LKb/?

Encrypt[e] / Decrypt[d]?
d
Message :
pYy LKb/?
Key :
10
fOo BAr/?
````
